# Vulnerable Transit API
A deliberately vulnerable transportation booking API in C.

## Purpose
Educational security research. Demonstrates memory safety 
vulnerabilities, injection flaws, and broken authentication 
at the systems level.

## Vulnerabilities Index
- V1: SQL Injection (CWE-89)
- V2: Buffer Overflow (CWE-121)
- V3: Hardcoded Secret (CWE-798)
- V4: No Token Expiry (CWE-613)
- V5: Format String (CWE-134)
- V6: Race Condition (CWE-362)
- V7: Use-After-Free (CWE-416)
